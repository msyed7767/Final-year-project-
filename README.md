# Tourism Trends & Forecasting Analysis of Pakistan

## Project Overview

This Final Year Project analyzes tourism trends in Pakistan with a special focus on the northern regions. The system provides interactive visualizations, tourism statistics, and forecasting using Python.

## Features

- Tourism data analysis
- Interactive dashboard
- Forecasting using Machine Learning
- Data visualization
- Responsive frontend
- Pakistan tourism insights

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Structure

```
Backend/
    app.py
    Data/

Frontend/
    index.html
    style.css
    app.js

Images/
```

## Dataset

The dataset contains tourism statistics collected from publicly available tourism sources.

## Installation

1. Clone the repository

```
git clone <repository-link>
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the application

```
python app.py
```
